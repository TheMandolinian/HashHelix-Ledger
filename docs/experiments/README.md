# HashHelix Experiment Archive

This directory contains the full archival record for all HashHelix experiments.
Each experiment is stored in its own folder and includes:

- PDF technical report
- Source code used to generate results
- Raw CSV outputs (when applicable)
- Figures (plots, phase-space diagrams, modular signatures)
- Experiment metadata

Structure:

exp01A — Small-N Sanity Check
exp01B — Medium-N Integrity Check
exp02 — Visual Signatures & Modular Invariant Discovery
exp03 — Periodicity & Drift Detection (upcoming)
...
